# 解决Ubuntu与Windows之间无法复制粘贴问题

1. 直接在命令行执行以下命令
```
sudo apt-get autoremove open-vm-tools	//卸载已有的工具
sudo apt-get install open-vm-tools		//安装工具open-vm-tools
sudo apt-get install open-vm-tools-desktop  //安装open-vm-tools-desktop
```

2. 重启Ubuntu系统即可
